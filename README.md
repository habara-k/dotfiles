# dotfiles

## USAGE

```sh
$ cd
$ git clone https://github.com/habara-k/dotfiles.git
$ cd dotfiles
$ chmod +x install.sh
$ ./install.sh

```

## Install

- HomeBrew
```sh
$ xcode-select --install
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

```

- git
```sh
$ brew install git

```

- zsh
```sh
$ brew install zsh

```

- neovim
```sh
$ brew install neovim

```

- tmux
```sh
$ brew install tmux

```
