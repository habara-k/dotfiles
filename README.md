# dotfiles

## USAGE

```sh
$ cd
$ git clone https://github.com/habara-k/dotfiles.git
$ cd dotfiles
$ chmod +x install.sh
$ ./install.sh

```

## Install

- HomeBrew
```sh
$ xcode-select --install
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

```

- git
```sh
$ brew install git

```

- zsh
```sh
$ brew install zsh
$ sudo vi /etc/shells
+ /usr/local/bin/zsh
$ chsh -s /usr/local/bin/zsh
```

- neovim
```sh
$ brew install neovim
$ brew install python3
$ pip3 install neovim

```

- tmux
```sh
$ brew install tmux

```

- zplug
```sh
$ brew install zplug

```

- fasd
```sh
$ brew install fasd

```

- pure-prompt
```sh
$ brew install npm
$ npm install --global pure-prompt

```
